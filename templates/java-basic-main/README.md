## Build

To compile the project and generate the JAR file, use the `build.sh` script:

```bash
./build.sh
```

This will:
- Compile all Java sources from `src/main/java`
- Copy resources from `src/main/resources`
- Generate an executable JAR in `target/build/`

### Run the application

To build and run the application directly:

```bash
./build.sh run
```

### Run tests

To build and execute the unit tests:

```bash
./build.sh test
```
